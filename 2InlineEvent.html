The Inline Event Handler

There are many different events that can occur and each one has a specific name. One of the most common events is a "click" event.

Event Handler code can be assigned at the element level. This code will be run for only that element and that is the only code that will be executed for that element event.

This is an example of the Button element running the code to jump to a different page when it is clicked with the mouse button.
<button onclick="window.location='1Javascript.html'"><- Javascript</button>

The "onclick" event is assigned the handler code "window.location= '1Javascript.html'". This jumps to the specified page when the button is clicked.

This type of event handler code should only used if there is only one or two code statements to be executed.
