Adding an Event

Event handlers can be assigned to an element in a script function instead of adding them with an inline event. Notice the AddEvents() function call at the end of the <body> element.

The addEventListener() function of the target element is used to make the assignment.

The target element can be referenced directly by its "id" name. 

  <head>
    <script>
      // Adds the required event handler assignment.
      function AddEvents()
      {
        title.addEventListener("click", TitleClick);
      }

      // The title click event handler.
      function TitleClick()
      {
        alert("Title was clicked.");
      }
    </script>
  </head>
  <body>
    <h3 id="title">Adding an Event</h3>

    <script>
      AddEvents();
    </script>
  </body>
