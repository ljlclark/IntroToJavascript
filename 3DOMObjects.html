DOM Objects

At the very top level of the DOM object tree is the "window" object. The window object represents the browser window where the HTML Document is displayed.

The window object contains the "document" object. The document object represents the Document Element which is the top-level element in an HTML page. The Document Element tag is <HTML>.

The document object contains the "head" object and "body" object. Notice how the object structure directly reflects the HTML Document element structure.
